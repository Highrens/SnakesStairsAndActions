# Snakes, Stairs and Actions

Небольшое веб-приложение на Next.js с генерацией случайных игровых событий для настольной игры в стиле "Змеи и лестницы".

## Что умеет сейчас

- Выдает случайное событие по кнопке "Действие!".
- Показывает варианты действий для события.
- Генерирует случайный результат выбранного варианта.
- Позволяет включить режим "убирать уже использованные события".

## Технологии

- Next.js 15 (App Router)
- React 19
- CSS modules + component CSS

## Локальный запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите dev-сервер:

```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm run start
```

## Структура

- `src/app` - layout и главная страница.
- `src/components` - UI-компоненты и основной игровой блок `ActionButton`.
- `src/constants/actions.js` - наборы игровых событий и вариантов исходов.
- `public` - статические ассеты (фон и изображения событий).
